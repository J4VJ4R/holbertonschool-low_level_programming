# 0x00. C - Hello, World

This project introduces the basics of C programming, including the compilation process, preprocessor directives, and basic C syntax.

## Requirements

- All files compiled on Ubuntu 20.04 LTS using gcc
- Compilation flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`
- All files end with a new line
- Code follows Betty style
- No use of `system` function
- No errors or warnings during compilation

## Tasks

### 0. Preprocessor
- **File**: `0-preprocessor`
- Script that runs a C file through the preprocessor and saves the result into another file
- The C file name is saved in the variable `$CFILE`
- Output is saved in the file `c`
