# holbertonschool-low_level_programming

This repository contains low-level programming projects and exercises completed as part of the Holberton School curriculum. All projects are written in C and follow the Betty coding style.

## Compilation

All files are compiled on Ubuntu 20.04 LTS using:
```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89
```

## Projects

- **0x00-hello_world**: Introduction to C programming - preprocessor, compiler, assembler, and basic C syntax
